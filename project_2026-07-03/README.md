# Daily Project - 2026-07-03

## Overview

Secure Password Generator

This project provides a cryptographically secure password generator utility, featuring both a Python class module and a command-line interface (CLI). It uses the built-in `secrets` module to ensure that the randomly selected characters are secure and less predictable than those generated by the standard `random` module.

### Features
* Customizable length (default 16).
* Optional inclusion/exclusion of lowercase, uppercase, digits, and symbols.
* Cryptographically secure shuffling using `secrets.SystemRandom().shuffle()`.

### Usage (CLI)
You can run the tool directly from your terminal:
```bash
python3 password_generator.py -l 24 --no-symbols
```

#### Arguments
* `-l`, `--length`: Set the desired password length (e.g., `-l 20`).
* `--no-lower`: Exclude lowercase letters.
* `--no-upper`: Exclude uppercase letters.
* `--no-digits`: Exclude numbers.
* `--no-symbols`: Exclude punctuation marks.

### Testing
To run the included unit test suite, make sure you are in the directory containing `test_password_generator.py` (or set your `PYTHONPATH`), and run:
```bash
PYTHONPATH=project_2026-07-03 python3 -m unittest project_2026-07-03/test_password_generator.py
```
